# OpenDrop Technical Mentoring Manual: Compiled PDF

This directory is reserved for the compiled PDF of the **10-Chapter OpenDrop Technical Mentoring Manual** generated from the LaTeX source code inside `/OpenDrop-TechDoc/`.

## 🚀 Double-Asset GitHub Strategy Instructions
To maximize visibility and demonstrate exceptional technical depth to hiring managers and recruiters:
1. Compile the LaTeX project in **Overleaf** using pdfLaTeX.
2. Download the resulting compiled PDF book.
3. Rename the file to `OpenDrop_Technical_Prep_Manual.pdf`.
4. Copy the PDF into this local directory: `/OpenDrop V2/docs/OpenDrop_Technical_Prep_Manual.pdf`.
5. Stage, commit, and push this PDF to your remote GitHub repository.
6. Pin your **Live Deployment URL** (from Netlify) at the top of your GitHub repository description, and point recruiters directly to `/docs/OpenDrop_Technical_Prep_Manual.pdf` within your main repository README.
